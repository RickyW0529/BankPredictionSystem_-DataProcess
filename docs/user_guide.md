# 银行预测数据处理系统 - 用户操作手册

## 一、环境准备

### 1. 安装 Python
- Windows 用户：访问 [https://www.python.org/downloads/](https://www.python.org/downloads/) 下载 Python 3.9 或更高版本，安装时**勾选 "Add Python to PATH"**。
- Mac 用户：系统通常自带 Python3，可在终端输入 `python3 --version` 检查。

### 2. 安装依赖
打开项目文件夹，运行以下命令：
```bash
pip install -r requirements.txt
```
或者双击 `start.bat`（Windows）/ `start.sh`（Mac/Linux），脚本会自动检查并安装依赖。

---

## 二、准备数据

### 文件夹结构
在项目根目录下创建 `raw_data` 文件夹，并在其中按频率建立子文件夹：

```
raw_data/
├── daily/          ← 日度数据（如股价、日度宏观指标）
├── monthly/        ← 月度数据（如 CPI、PPI、PMI）
└── quarter/        ← 季度数据（如 GDP、居民收入）
```

### 数据格式要求
- **文件类型**：CSV (`.csv`) 或 Excel (`.xlsx` / `.xls`)
- **首列必须是日期列**，列名可以是 `指标名称`、`date`、`日期`、`DATA_DATE` 等
- **支持的日期格式**：
  - `2019/1/1` 或 `2019-01-01`
  - `2019年1月`
  - `19-Jan`
  - `20190101`

### 示例
```
raw_data/
├── daily/
│   └── 日度宏观数据.csv
├── monthly/
│   └── 月度宏观数据.csv
└── quarter/
    └── 季度宏观数据.csv
```

## 三、启动系统

### Windows
双击项目文件夹中的 **`start.bat`**，稍等几秒，浏览器会自动打开操作界面。

### Mac / Linux
在终端进入项目文件夹，运行：
```bash
./start.sh
```
浏览器会自动打开 `http://localhost:8501`。

## 四、操作界面说明

界面分为左右两部分：左侧是**参数配置栏**，右侧是**主操作区**。

### 1. 参数配置（左侧栏）

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **运行模式** | "仅合并数据" 不需要目标变量；"完整特征工程" 需要额外提供目标文件 | 仅合并数据 |
| **缺失值阈值** | 缺失率超过此百分比的列会被删除 | 60% |
| **使用 tsfresh** | 是否使用 tsfresh 进行自动特征提取（仅完整模式可用） | 是 |
| **FDR 水平** | 特征筛选的显著性水平 | 0.05 |
| **时间窗口** | tsfresh 回看的最小/最大月数 | 3 / 6 |
| **最大特征比例** | 特征数上限（相对于样本数） | 0.6 |

### 2. 数据准备（主区域上半部分）
- 系统会自动检测 `raw_data` 目录是否存在
- 若不存在，点击 **"打开 raw_data 文件夹"** 按钮，系统会自动创建目录并用资源管理器打开
- 若存在，会显示文件列表和每个文件的前 5 行预览

### 3. 运行处理（主区域中部）
确认数据无误后，点击大按钮 **"开始处理"**。
- 运行过程中会显示实时日志
- 处理完成后会提示成功

### 4. 下载结果（主区域下半部分）
处理完成后，界面会列出所有输出文件，点击 **"下载"** 按钮即可保存到本地。

| 输出文件 | 说明 |
|----------|------|
| `features_merged.csv` | 合并后的月度数据（仅合并模式） |
| `train_pca.csv` | PCA 降维后的训练数据（完整模式 + tsfresh） |
| `train_no_pca.csv` | 未做 PCA 的特征数据（完整模式 - no-tsfresh） |

## 五、常见问题

### Q1: 双击 start.bat 闪退？
- 通常是未安装 Python 或 Python 未加入环境变量。请重新安装 Python 并勾选 "Add Python to PATH"。

### Q2: 提示 "No date column found"？
- 检查 CSV 首列是否是日期，列名是否包含 `指标名称`、`date`、`日期` 等关键字。

### Q3: 处理结果只有几列，很多数据被删除了？
- 调大**缺失值阈值**（如 80 或 100），或检查原始数据是否缺失过多。

### Q4: Mac 运行 start.sh 提示 Permission denied？
- 先在终端执行 `chmod +x start.sh`，再运行 `./start.sh`。
